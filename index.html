이메일 서명 생성기

서명을 편집한 후 HTML을 복사하여 사용하는 메일 서비스의 서명 설정에 붙여넣기 하세요.

C-level: COO(최고운영책임자), CMO(최고마케팅책임자), CTO(최고기술책임자), CEO(최고경영자)
직원 정보를 불러온 뒤에도 영문 이름은 직접 수정할 수 있습니다.
* 실제 이메일에서의 모습을 미리 확인하세요.
▣ 네이버 메일 서명 설정 방법 1. HTML 복사하기 버튼 클릭
2. 네이버 메일 → 환경설정 → 서명 관리
3. HTML 편집 모드로 전환
4. 복사한 코드 붙여넣기 (Ctrl+V) → 저장

▣ 다음 메일 서명 설정 방법 1. HTML 복사하기 버튼 클릭
2. 다음 메일 → 환경설정 → 서명
3. 서명 추가 또는 기존 서명 편집
4. HTML 입력/편집 화면에 붙여넣기 → 저장

▣ Gmail 서명 설정 방법 1. HTML 복사하기 버튼 클릭
2. Gmail → 설정 → 모든 설정 보기 → 기본설정
3. 서명 항목에서 새 서명 만들기 또는 기존 서명 선택
4. 서명 입력 칸에 붙여넣기 (Ctrl+V) → 변경사항 저장
